# STEV - Android Coding Agent

An intelligent coding agent for Android development powered by GitHub Copilot.

## Overview
STEV is designed to assist with Android app development, providing intelligent code suggestions, debugging support, and best practices guidance.

## Getting Started
1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle files
4. Build and run the app

## Project Structure
- `app/` - Main Android application module
- `agent/` - Coding agent implementation

## Requirements
- Android Studio Arctic Fox or later
- Java 11 or higher
- Gradle 7.0+
